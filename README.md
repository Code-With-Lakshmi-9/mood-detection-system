# 😊 Mood Detection and Suggestion System

This is a simple command-line based Mood Detection and Suggestion System built using Python.

It helps users identify their current mood and gives helpful suggestions to improve or manage their feelings.

## 🚀 Features
- Detects user mood based on input
- Supports multiple moods:
  - Happy 😊
  - Sad ☹️
  - Angry 😠
  - Anxious/Worried 😥
  - Tired 🥱
  - Bored 😒
  - Confident 😎
  - Lonely 💔
- Provides suggestions based on selected mood
- Interactive user input system

## 🛠️ Technologies Used
- Python (Basics)

## ▶️ How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Open terminal or command prompt
4. Run the program:

```bash
python mood.py
